# saticatapibnr
Small App That Implements CAT API, But Not Really

### Requirements

Python 3.6+

Connexion 2.0
Install with:
```
pip install connexion[swagger-ui]
```

Tornado 6.0.4
Install with:
```
pip install tornado
```

### Run

```
python app.py
```

### Swagger UI

http://localhost:8080/ims/cat/v1p0/ui/
